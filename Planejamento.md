# FNFIPF - Planejamento

## Visão
App estilo Duolingo para praticar inglês, integrado ao Cerebro.

## Features Principais (Baseadas no Duolingo)

### 1. Sistema de Lições
- [ ] Lições diárias (Daily10)
- [ ] Lições por tópicos (grammar, vocab, conversation)
- [ ] Revisão de erros (mistakes)

### 2. Gamificação
- [ ] XP points
- [ ] Streak (dias consecutivos)
- [ ] Hearts (vidas)
- [ ] Levels
- [ ] Leaderboards (futuro)

### 3. Prática
- [ ] Multiple choice
- [ ] Tradução PT → EN
- [ ] Speaking practice (texto)
- [ ] Listening practice
- [ ] Dictation

### 4. Recursos
- [ ] Stories em inglês
- [ ] Common Phrases
- [ ] Mistakes database
- [ ] Phase Builders

### 5. Gamificação Social (futuro)
- [ ] Friends
- [ ] Quests
- [ ] Challenges

## Tecnologias
- Frontend: VS Code (HTML/CSS/JS) ou app mobile
- Backend: Pode rodar local (JS) ou API
