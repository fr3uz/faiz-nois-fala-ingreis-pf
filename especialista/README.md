# FNFIPF - Especialista IA

O Especialista é seu tutor pessoal de inglês estilo Duolingo.

## Quem é?

- Nome: **Professor James** (ou "James")
- Personalidade: Amigável, paciente, motivacional como o Duo
- Objetivo: Te ajudar a aprender inglês de verdade

## Funcionalidades

### 1. Explicação de Erros
Quando você erra uma questão, o especialista explica:
- **Por que** a resposta está errada
- **Qual** a regra grammatical
- **Exemplo** prático

### 2. Dicas em Tempo Real
- Dá dicas antes de você responder (se pedir)
- Explica expressões idiomáticas
- Ensina vocabulário relacionado

### 3. Modo Conversa
- Prática de diálogos (entrevistas, reuniões, etc)
- Perguntas abertas para vocêResponder em inglês
- Corrige sua gramática

### 4. Feedback Personalizado
- Baseado nos seus erros gravados
- Foca nas áreas fracas
- Sugere temas para praticar

### 5. Motivação
- Diz frases quando você acerta/erra
- celebra metas
- Mantém o streak

## Como Usar

1. No perfil, clique em "Falar com Especialista"
2. Ou clique em "❓ Ajuda" durante uma questão
3. O especialista abre em uma aba/chat

## Integração com Groq API

O especialista usa a Groq API (Llama 4 Scout) para gerar explicações personalizadas em português.
